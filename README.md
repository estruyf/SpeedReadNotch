# SpeedReadNotch

Native macOS Speed Reading application for the notch.

## Architecture
Built with Swift and SwiftUI, following the `Textream` infrastructure.

## Setup
1. Open Xcode and select **"Create a new Xcode project"**.
2. Choose **macOS** -> **App**.
3. Name the product **"SpeedReadNotch"**.
4. Set the Interface to **SwiftUI** and Language to **Swift**.
5. Save the project inside `/Users/eliostruyf/Developer/tauri/speedread-notch/`.
6. Once created, **replace** the default files with the source files provided in
   the `SpeedReadNotch/SpeedReadNotch` folder (drag and drop them into the
   project navigator).
2. Ensure Signing & Capabilities are configured.
3. Build and Run.

See `plan.md` for development roadmap.
